# スピナー・ドロップ - Spinner Drop

回転するスピナーの隙間にボールを落とすタイミングパズルゲーム

## 特徴

- 100ステージのタイミングチャレンジ
- シンプルだけど中毒性が高い
- 美しいグラデーション表示
- 物理演算で気持ちいい

## 遊び方

1. 画面をタップしてボールを落とす
2. 回転するスピナーの隙間を通過させる
3. 3回成功でステージクリア
4. 棒に当たるとボールが消える

## 技術スタック

- Capacitor 7
- iOS 14以上
- Canvas API（物理演算）
- AdMob広告（インタースティシャル）
- LocalStorage（進行状況保存）

## 開発

```bash
npm install
npx cap add ios
npx cap sync ios
```
