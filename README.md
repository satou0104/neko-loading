# ネコローディング - Neko Loading

猫の手からスピナーを守るゲーム

## 特徴

- 30ステージのタップチャレンジ
- 4種類の猫の手を見分けてタップ
- 白黒スタイリッシュなスピナー
- ステージが進むほど猫の手が速くなる

## 遊び方

1. 画面上のスピナーに向かって猫の手が伸びてくる
2. 画面下の猫の手ボタンから同じ種類をタップ
3. 正しくタップすると猫の手が引っ込む
4. スピナーに到達されるとローディングが減る
5. 100%まで守り切ればステージクリア

## 技術スタック

- Capacitor 7
- iOS 14以上
- Canvas API
- AdMob広告（インタースティシャル）
- LocalStorage（進行状況保存）

## 開発

```bash
npm install
npx cap add ios
npx cap sync ios
```
